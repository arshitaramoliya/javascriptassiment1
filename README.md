# JavaScript — Objects, Arrays & Functional Methods

JavaScript assignment focused on working with structured data — objects, arrays, and functional array methods like `filter()`.

## Topics Covered

| File | Topic |
|---|---|

| `two.js` | Array manipulation exercises |
| `three.js` | Object property access and mutation |
| `four.js` | Nested data structures |

## Key Example

`one.js` defines an employee dataset as an array of objects and uses `Array.filter()` to query records by property — the same pattern used in real-world data processing and API response handling.
